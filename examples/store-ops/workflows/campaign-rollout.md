# 檔期上線流程

## 觸發
我說「啟動XX檔期」或收到總部檔期公告要佈達

## 步驟

### 1. 讀懂公告
- 我貼上總部的檔期公告
- 整理出：時間、主推品、促銷機制、門市需配合事項

### 2. 庫存檢查
- 用 .agents/skills/inventory-alert/ 檢查主推品的各店庫存
- 如果低於安全庫存 → 立即產出補貨建議
- 👉 停下來讓我確認庫存狀況

### 3. 門市佈達
- 用 .agents/skills/campaign-rollout/ 產出：
  - LINE 群白話版通知
  - 各店差異化提醒
  - 執行追蹤表
- 👉 停下來讓我確認內容

### 4. 追蹤機制
- 設定追蹤節點：
  - 佈達後 D+1：確認各店已讀已回覆
  - 佈達後 D+3：各店陳列完成照片回傳
  - 檔期開始 D+1：首日業績快報
  - 檔期結束 D+3：檔期檢討報告

## 輸出
- LINE 通知 → 顯示在對話中讓我複製
- 追蹤表 → outputs/YYYY-MM/[檔期名稱]/追蹤表.md
- 補貨建議（如有）→ outputs/YYYY-MM/[檔期名稱]/補貨建議.md
