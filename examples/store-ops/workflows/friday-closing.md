# 週五收工流程

## 觸發
我說「寫週報」「週五了」「收工」

## 步驟

### 1. 彙整數據
- 我貼上本週各店的數據彙整
- 如果這週有日報 → 自動從 outputs/ 裡找本週的日報來彙整

### 2. 週報產出
- 用 .agents/skills/weekly-report/ 產出完整週報
- 格式符合陳經理的偏好（先結論、表格為主、一頁以內）
- 👉 停下來讓我確認

### 3. 下週規劃
- 根據本週數據，建議下週的重點行動：
  - 哪家店需要巡？
  - 有沒有排班或庫存問題要提前處理？
  - 有沒有客訴案件待追蹤？
  - 有沒有檔期要準備？

### 4. 經驗更新
- 👉 問我：「這週有沒有什麼心得要加到做事原則裡？」
- 如果有 → 幫我更新 CLAUDE.md 的做事原則區塊

## 輸出
- 週報 → outputs/YYYY-MM/週報-第N週.md
- 下週計畫 → 顯示在對話中
