# 巡店流程

## 觸發
我說「今天巡了XX店」「整理巡店紀錄」

## 步驟

### 1. 收集筆記
- 我會貼上巡店時的碎片筆記（手打、語音轉文字、照片描述混在一起）
- 👉 如果資訊不足，問我補充：環境狀況？陳列？人員？跟店長聊了什麼？

### 2. 整理紀錄
- 用 .agents/skills/store-inspection/ 產出完整巡店報告
- 自動評分（根據描述估算）
- 改善項目要有負責人和期限

### 3. 確認
- 👉 給我確認評分和改善項目
- 我可能會調整分數或補充內容

### 4. 發送版本
- 產出一份給店長的版本（語氣建設性，先肯定再改善）
- 產出一段 LINE 訊息版（精簡，只講需要改的和期限）

## 輸出
- 完整巡店報告 → outputs/YYYY-MM/巡店-[門市]-MM-DD.md
- 店長版 + LINE 版 → 顯示在對話中讓我複製
