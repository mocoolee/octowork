# 門市日報分析器

## 執行前
1. 讀 company.md → 各店月目標、來客數目標、客單價目標
2. 讀 CLAUDE.md → 數據判讀原則

## 輸入
使用者會貼上各門市的昨日數據，格式可能不固定（Excel 截圖、手打數字、LINE 訊息轉貼都有可能）。需要的欄位：
- 各門市營業額
- 來客數
- 客單價（如果沒給就用營業額÷來客數算）

## 產出格式

```markdown
# 日報 ── YYYY/MM/DD（週X）

## 昨日A區總覽
| 指標 | 昨日 | 月累計 | 月目標 | 達成率 |
|------|------|--------|--------|--------|

## 各店表現
| 門市 | 營業額 | vs 目標日均 | 來客 | 客單價 | 狀態 |
|------|--------|------------|------|--------|------|

（狀態：🟢 達標 / 🟡 接近 / 🔴 落後）

## 異常與行動
- 🔴 [門市名]：[異常描述]，建議 [具體行動]
- ...

## 今日重點關注
1. ...
```

## 判斷邏輯
- 日營業額 = 月目標 ÷ 當月營業天數（通常 30 天）
- 達成率 ≥ 100%：🟢
- 達成率 90-99%：🟡
- 達成率 < 90%：🔴 → 必須附上建議行動
- 客單價偏離目標 ±10% 以上要特別標註
