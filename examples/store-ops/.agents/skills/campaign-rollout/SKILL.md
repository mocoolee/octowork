# 檔期佈達助手

## 執行前
1. 讀 company.md → 各店特性
2. 讀 products.md → 涉及的商品資料
3. 讀 CLAUDE.md → 檔期處理原則

## 輸入
總部發的檔期公告（通常是官方版、比較硬的文字）

## 產出

### 1. 門市白話版通知（發 LINE 群用）
```
📢 [檔期名稱] 執行重點

🗓 時間：MM/DD - MM/DD
🎯 主推品：...
💰 促銷內容：...

📋 你要做的事：
□ [具體動作 1]（完成期限）
□ [具體動作 2]（完成期限）
□ [具體動作 3]（完成期限）

📸 完成後拍照回傳到群組確認
❓ 有問題找子翔
```

### 2. 各店差異化提醒
（根據各店特性，提醒不同的重點）
- 東區店：人流量大，注意 ___
- 板橋店：百貨配合事項 ___
- 新莊店：旗艦店要 ___
- 三重店：新店需要額外 ___

### 3. 執行追蹤表
| 門市 | 陳列完成 | 價卡更換 | 人員佈達 | 庫存確認 | 照片回傳 |
|------|---------|---------|---------|---------|---------|
| 東區 | □ | □ | □ | □ | □ |
| 板橋 | □ | □ | □ | □ | □ |
| 新莊 | □ | □ | □ | □ | □ |
| 三重 | □ | □ | □ | □ | □ |

## 原則
- 總部公告通常很長很官方，門市人員看不完 → 精簡成重點
- 「你要做的事」每一條都要是具體動作，不要寫「請配合執行」
- 每家店的提醒要根據那家店的實際狀況客製化
