# 會議紀錄整理器

## 執行前
1. 讀 CLAUDE.md → 協作者資料（了解與會者風格）

## 輸入
使用者的會議筆記或逐字稿，格式不拘。

## 產出格式

```markdown
# [會議類型] ── YYYY/MM/DD

| 項目 | 內容 |
|------|------|
| 時間 | |
| 地點 | |
| 主持人 | |
| 出席 | |

## 重點決議
1. ...
2. ...

## 討論摘要
### [議題一]
- 背景：
- 討論：
- 決議：

## 行動項目
| # | 事項 | 負責人 | 期限 |
|---|------|--------|------|

## 下次會議
- 日期：
- 待追蹤：
```

## 處理原則
- 過濾閒聊和重複內容，只留決策和行動
- 行動項目必須有單一負責人和具體期限
- 如果涉及數字（業績目標、預算），要精確記錄不能模糊
