# 週報流程

## 觸發
我說「寫週報」

## 步驟

### 1. 貼數據
- 我把本週各管道的數據貼進來（GA4、廣告後台、MailerLite、Shopify）
- 格式不限，什麼樣子都可以

### 2. 產出完整週報
- 用 weekly-report Skill
- 自動跟上週比較、標出異常、分析最佳和最差
- 👉 停下來讓我確認內容，特別是「異常說明」和「下週計畫」

### 3. 產出 LINE 精簡版
- 把週報濃縮成 3 行重點，給 David 看的版本
- 格式：本週營收 $X（+Y%）/ ROAS X.X / 最大亮點一句話

### 4. 回顧
- 問我：這週有沒有新的經驗值得加到做事原則？
- 例如：某個素材意外爆了、某個時段投放效果特別好

## 輸出
- 完整版：outputs/YYYY-MM/週報-第N週.md
- LINE 精簡版：可以直接貼給 David
