# SEO 內容產出流程

## 觸發
我說「寫 SEO 文章」或直接給一個目標關鍵字

## 步驟

### 1. 確認關鍵字
- 確認目標關鍵字（一篇只打一個）
- 讀 company.md → 了解 SEO 管道現況
- 讀 products.md → 確認可以內連結到哪些產品頁

### 2. 產出大綱
- 用 seo-outline Skill
- 👉 停下來讓我確認大綱結構和預估字數

### 3. 撰寫完整文章
- 根據確認的大綱寫完整文章（≥ 1,500 字）
- 語氣符合品牌調性（溫暖、有生活感）
- 👉 停下來讓我確認內容

### 4. SEO 自我檢查
- Title Tag 有沒有包含關鍵字（< 60 字）
- Meta Description 有沒有吸引點擊的理由（< 160 字）
- 前 100 字有沒有出現主關鍵字
- 內部連結 ≥ 3 個
- H2/H3 結構清楚

### 5. 產出上架格式
- 整理成 Shopify Blog 可以直接貼上的格式
- 附上建議的 URL slug

### 6. 要不要順便發社群？
- 問我：要不要用 social-post Skill 順便產一篇導流貼文
- 如果要 → 跑 social-post Skill

## 輸出
- 完整文章：outputs/YYYY-MM/SEO-[關鍵字].md
- 社群導流貼文（如果有）：存在同一資料夾
