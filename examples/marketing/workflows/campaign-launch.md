# 檔期上線流程

## 觸發
我說「啟動 XX 檔期」

## 步驟

### 1. 確認檔期資訊
- 讀 products.md → 確認主推品、價格、促銷方案
- 讀 company.md → 確認客群和品牌語氣
- 跟我確認：檔期時間、預算上限、主要目標（拉新 / 回購 / 清庫存）

### 2. 產出 Landing Page 文案
- 用 landing-page Skill
- 👉 停下來讓我確認文案，特別是 Hero 大標和 CTA

### 3. 產出廣告文案
- 用 ad-copy Skill → 產出 5 組不同角度
- 👉 停下來讓我選要用哪幾組
- 附上投放建議（預算、時段、素材形式）

### 4. 產出 EDM
- 用 edm-writer Skill
- 👉 停下來讓我確認主旨行和內文

### 5. 產出社群貼文
- 用 social-post Skill → IG + FB + Stories
- 👉 停下來讓我確認

### 6. 產出設計 Brief
- 整理給設計師阿倫的需求：需要哪些素材、尺寸、文案重點、deadline
- 格式要具體，阿倫不喜歡模糊的 brief

### 7. 產出上線 Checklist
- 投放設定確認（受眾、預算、排程）
- UTM 參數設定
- LP 上線確認
- EDM 排程確認（週二早上 10 點）
- 社群排程確認

## 輸出
- 完整版：存到 outputs/YYYY-MM/[檔期名稱]/
- Checklist 精簡版：可以直接貼到 LINE 跟自己 check
