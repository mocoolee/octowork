# 廣告文案生成器

## 執行前
1. 讀 products.md → 產品賣點、價格、歷史最佳素材
2. 讀 company.md → 品牌語氣、客群 persona
3. 讀 CLAUDE.md → 廣告原則、文案原則

## 輸入
使用者提供：
- 要推的產品（或從 products.md 挑）
- 投放目的（拉新 / 再行銷 / 促銷 / 品牌曝光）
- 目標客群（安心媽媽 / 質感獨居族 / 不指定）
- 平台（Meta / Google / 不指定）

## 產出格式

針對同一產品，產出 5 組不同切角的文案：

```markdown
# 廣告文案 ── [產品名] × [投放目的]

## 角度一：痛點型
- **Hook：** （一句話戳中痛點，建議用問句，CTR 較高）
- **Body：** （先講使用情境，再帶出產品怎麼解決）
- **CTA：** （明確行動指令）

## 角度二：情境型
- **Hook：**
- **Body：** （描繪使用場景，讓人代入）
- **CTA：**

## 角度三：社會證明型
- **Hook：** （數字或他人推薦開頭）
- **Body：** （真實回饋 + 產品特色）
- **CTA：**

## 角度四：好奇型
- **Hook：** （製造資訊缺口）
- **Body：** （揭曉答案 + 產品連結）
- **CTA：**

## 角度五：急迫型
- **Hook：** （限時/限量/即將結束）
- **Body：** （促銷方案 + 為什麼現在要買）
- **CTA：**

## 投放建議
- 建議測試預算：$500/天，跑 3 天
- 優先測：角度 X（根據歷史數據判斷）
- 素材形式建議：影片 / 靜態圖 / 輪播
```

## 品質檢查
- [ ] 每組 Hook 都不一樣，角度有明確區隔
- [ ] 永遠先寫使用情境，再寫產品規格
- [ ] 至少一組用問句開頭（CTR 較高 22%）
- [ ] 促銷用「買二送一」而非「打折」的表述（轉換率高 40%）
- [ ] 語氣符合品牌調性（溫暖真誠，不用「獨家黑科技」「買到賺到」）
- [ ] CTA 具體明確（「點這裡看成分」比「了解更多」好）
