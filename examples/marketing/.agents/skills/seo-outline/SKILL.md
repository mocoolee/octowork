# SEO 文章大綱產生器

## 執行前
1. 讀 products.md → 可以內連結到哪些產品頁
2. 讀 company.md → 品牌定位、客群關心的議題
3. 讀 CLAUDE.md → SEO 原則

## 輸入
使用者提供：
- 目標關鍵字（一篇只打一個）
- 文章目的（導購 / 品牌內容 / 長尾流量）
- 有沒有想涵蓋的子題

## 產出格式

```markdown
# SEO 大綱 ── 目標關鍵字：[關鍵字]

## 文章設定
- **標題：** （含關鍵字，50 字以內）
- **Meta Description：** （含關鍵字，150 字以內，要有吸引點擊的理由）
- **URL Slug：** /blog/[slug]
- **預估字數：** 1,800 字

## 文章結構

### H2：[段落標題]（約 300 字）
- 要點 1
- 要點 2
- 💡 前 100 字內出現主關鍵字

### H2：[段落標題]（約 400 字）
- 要點 1
- 要點 2
  - H3：[子段落]（約 200 字）

### H2：[段落標題]（約 400 字）
- 要點 1
- 要點 2

### H2：[段落標題]（約 300 字）
- 要點 1
- 要點 2

### H2：結論（約 200 字）
- 重申重點
- CTA：導向產品頁

## 內部連結建議（至少 3 個）
1. [錨點文字] → [連結目標]
2. [錨點文字] → [連結目標]
3. [錨點文字] → [連結目標]
```

## 品質檢查
- [ ] 一篇只打一個主關鍵字
- [ ] 前 100 字出現主關鍵字
- [ ] 總字數 ≥ 1,500 字
- [ ] H2/H3 層級清楚，有目錄結構
- [ ] 內部連結 ≥ 3 個
- [ ] Meta Description 有吸引點擊的理由，不只是摘要
