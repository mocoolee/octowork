# 社群貼文生成器

## 執行前
1. 讀 products.md → 產品賣點、適合的內容角度
2. 讀 company.md → 品牌語氣、客群、管道現況
3. 讀 CLAUDE.md → 文案原則

## 輸入
使用者提供：
- 主題或產品
- 平台（IG / FB / 全部）
- 貼文類型（產品推薦 / 知識分享 / 活動宣傳 / 日常互動）

## 產出格式

```markdown
# 社群貼文 ── [主題]

## IG 貼文
**Caption：**
（開頭要有 hook，第一行決定別人要不要點「更多」）

（內文 3-5 行，情境 > 規格）

（CTA：引導互動或點連結）

**Hashtag（15-20 個）：**
#品牌標籤 #產品標籤 #情境標籤 #大流量標籤

---

## FB 貼文
（比 IG 長一點，可以說故事）
（開頭一樣要有 hook）
（語氣可以更口語、更像聊天）

---

## Stories 短文案（3 張）
1. 第一張：hook（問句或驚人事實）
2. 第二張：內容重點
3. 第三張：CTA（往上滑 / 私訊我們）
```

## 品質檢查
- [ ] 每篇第一行都有 hook（問句式開頭 CTR +22%）
- [ ] 語氣像 32 歲設計師媽媽跟朋友聊天
- [ ] 不出現「獨家黑科技」「買到賺到」「限時搶購」
- [ ] IG 和 FB 的內容有差異，不是複製貼上
- [ ] Hashtag 混合品牌標籤和大流量標籤
- [ ] Stories 三張有節奏感（hook → 內容 → 行動）
